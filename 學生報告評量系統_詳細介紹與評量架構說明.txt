================================================================================
           學生報告評量系統 (PBL 多元學習成就評量) — 詳細介紹與評量架構說明書
================================================================================

壹、系統整體簡介與設計理念
--------------------------------------------------------------------------------
「學生報告評量系統」是一套專為專題導向學習（Project-Based Learning, PBL）與期末成果發表而設計的多元評量與學習成就回饋系統。

傳統的小組報告評量常面臨三個核心挑戰：
1. 單一主觀性：僅由教師在台下打分，難以看見各小組內部成員間的分工與實質付出。
2. 缺乏互助激勵：小組報告易淪為少數人全包，或出現搭便車（Free-riding）現象。
3. 質性與量化分離：僅有分數缺乏具體稱讚與建議，或質性文字零散不易彙整。

本系統透過「遊戲化闖關體驗（三關卡）」，巧妙融合：
- 自我評量（Self-Assessment）：學生後設認知與自我省思。
- 同儕評量（Peer-Assessment）：組內欣賞協作與閃光時刻驗證。
- 跨組標竿學習（Cross-Group Appreciation）：他組星探發掘與觀摩表揚。
- 專家教師評定（Expert Assessment）：教師全域視角綜合把關。

達成「量化計分嚴謹公正、質性回饋具體溫暖」的 360 度立體評量架構。


貳、三大闖關關卡與核心評量向度
--------------------------------------------------------------------------------
系統設計三個循序漸進的關卡，分別對應不同的學習目標與評量規準：

一、【關卡一：學生自評表】（個人表現與反思評量）
- 評量目的：培養學生對自身任務參與度、專題素養與反思表達的客觀審視能力。
- 量化向度（李克特五點量表，共 7 大題，原始滿分 35 分）：
  1. 資料蒐集：使用可信來源、整理內容資訊
  2. 資料整理：去蕪存菁、用自己的文字產出（避免抄襲）
  3. 小組討論：尊重發言、積極貢獻想法
  4. 簡報討論：努力貢獻心力、尊重組員想法、共同呈現最佳內容
  5. 簡報製作：盡力完成負責部份、適時支援與協助其他組員
  6. 口頭報告：有條有理、抑扬頓挫、音調與台風適中
  7. 解謎與活動：攤位與活動當天堅守崗位、機動支援

- 評量規準（5 級分行為特徵描述）：
  - 5 分（積極用心）：全力以赴、表現卓越，不僅高品質達成職責，更能主動引導並協助同伴。
  - 4 分（表現良好）：態度認真、穩定可靠，準時且落實完成所有工作與討論標準。
  - 3 分（表現平平）：達成基本門檻，但在主動性、表達或協助方面尚有進步空間。
  - 2 分（仍需努力）：被動參與，偶有遲交或工作完成度不足，需組員提醒與敦促。
  - 1 分（勉強做到）：參與度偏低，未能充分承擔相應責任。

- 質性與反思向度：
  - 榮譽自我定位：從 6 大團隊勳章中選擇一枚最符合自己本學期表現的專長榮譽勳章。
  - 壓軸質性大哉問（引導式反思，嚴格驗證至少 5 字以上）：
    (1) 整個課程中，我最喜歡的部分是？
    (2) 透過課程學習，我覺得我學到了哪些能力？


二、【關卡二：組內互評表】（團隊協作與優勢鑑賞）
- 評量目的：轉化傳統互評容易出現的「比較與傷害」，引導學生以「尋找團隊超能力」的賞識心態，表揚每位夥伴。
- 量化與質性結合機制：
  - 雙勳章與彈性給分機制：
    對小組內每一位成員皆可給予一枚「必填第一勳章」與一枚「選填第二勳章」，每枚勳章對應給予 3 分、4 分或 5 分加分。
  - 六大團隊超能力勳章與特質定位：
    1. 資料淘金客：超會查資料、抓重點（探究專長）
    2. 進度鬧鐘：小組時空旅人，提醒截止時間（進度管理）
    3. 視覺魔法師：簡報美化與排版救星（美感排版）
    4. 氣場麥克風：上台發表清楚、不怯場（口頭發表）
    5. 腦洞發明家：點子王，解謎題目超有創意（創新思維）
    6. 溫暖神補位：團隊黏著劑，默默撿起沒人做的事（互助協作）
  - 具體閃光時刻（質性回饋強制驗證）：
    每給予一位夥伴評價，務必填寫至少 5 字以上的「具體閃光時刻」作為客觀佐證。

- 隱藏大招：組內 MVP 投票與真情告白
  - 每人推舉一位全組公認的「本組年度隱藏 MVP」。
  - 填寫「想對 MVP 說的一句感謝」。
  - 獲選 MVP 者，自動獲得教師後台設定的 MVP 額外加分（預設每票 +1 分，可自訂）。


三、【關卡三：他組星探推薦】（跨組標竿鑑賞與互動）
- 評量目的：打破「只看自己組別」的框架，引導學生在活動日走訪各組攤位與簡報，發掘傑出楷模。
- 三大星探榮譽獎項：
  1. 簡報吸睛王（台上的靈魂人物）
  2. 金頭腦出題王（解謎題目的設計鬼才）
  3. 熱情神關主（活動當天的最佳服務生）
- 加分回饋：獲別組星探提名推薦者，自動計入星探推薦加分（預設每次 +2 分，可自訂）。


參、質性與量化評量融合分析模型
--------------------------------------------------------------------------------
本系統讓「數字有依據，文字有溫度」：

[ 自我評量 ]
- 量化指標：7 題五點量表得分（滿分 35 分），自動依公式換算為 100 分制百分比。
- 質性指標：自給優勢定位勳章、最喜歡的課程環節反思、學到的能力自我盤點。

[ 組內互評 ]
- 量化指標：組員贈予勳章分數加總（每位組員給出 3~10 分）+ MVP 得票數累計加分。
- 質性指標：給予每位夥伴的具體閃光時刻描述、給予 MVP 的致謝真情留言。

[ 跨組鑑賞 ]
- 量化指標：獲得他組星探推薦總次數乘以單次設定分數。
- 質性指標：簡報吸睛王 / 出題王 / 神關主之具體推薦佐證描述。

[ 專家教師評量 ]
- 量化指標：教師綜合給定 0 ~ 100 分。
- 質性指標：教師根據全班各向度表現進行全域指導與總評。


肆、綜合成績運算公式與權重機制
--------------------------------------------------------------------------------
系統教師後台支援動態調整三方權重，分數即時自動運算：

1. 各向度運算
- 自評百分制 = (自評 7 題得分總計 / 35) x 100
- 他評加分總和 = 收到組員給予的勳章加分 + (MVP票數 x MVP單票加分) + (星探推薦次數 x 星探單次加分)
- 老師綜合評分 = 老師給定分數 (0 ~ 100 分)

2. 最終綜合成績公式
預設權重：自評 30%、他評 20%、老師 50%（百分比可在教師後台自訂儲存）：
最終綜合分數 = (自評百分制 x 30%) + (他評加分總和 x 20%) + (老師評分 x 50%)


伍、教師專屬後台與評量報表應用
--------------------------------------------------------------------------------
教師專屬後台分為五大功能頁籤：
1. 成績結算與打分：全班座號、組別、自評百分制、他評加分總和、最終分數總覽與老師打分輸入區。
2. 學生填寫紀錄（輸出明細）：完整查看學生填寫的自評各題分數、反思文字、給組員的互評與閃光時刻、MVP 推舉。
3. 學生收到評價（獲得回饋明細）：彙整學生被給予的勳章、分數、閃光時刻留言、MVP 感謝與他組星探推薦。
4. 名單管理：小組與座號管理、雲端同步套用。
5. 計分權重設定：調整自評、他評、老師評分百分比及加分規則，並可設定學生首頁登入密碼。

【報表一鍵匯出功能（支援 UTF-8 BOM CSV / Excel 雙格式）】
匯出總表共含 28 個全面性欄位，方便後續學期總結與歷程留存：
- 綜合成績概覽：座號、組別、最終綜合分數、老師評分、自評百分制、他評加分總和、MVP得票數、自給勳章。
- 自評輸入明細：自評總分與 1~7 題給分、兩大質性反思。
- 他評輸入明細：給予夥伴之互評紀錄（對象/勳章/分數/閃光時刻）、推舉 MVP 與感謝文字、三大星探推薦。
- 收到回饋明細：收到組內勳章與分數、收到閃光時刻留言明細、收到 MVP 感謝與他組星探推薦理由。


陸、學生首頁登入驗證與測試帳號機制
--------------------------------------------------------------------------------
1. 一般學生登入（雙重驗證）：
   - 學生需輸入「座號（1~30號）」與「登入密碼（預設為 c1723，教師可在後台自訂修改）」方能進入任務總部。
2. 專屬體驗測試帳號（31 號）：
   - 於座號欄輸入「31」號為系統體驗與測試帳號，無須登入密碼即可進入。
   - 測試帳號填寫的任何自評或互評資料皆不會上傳至資料庫，亦不會列入教師後台的成績與報表紀錄。
