# 學生報告評量系統 — 詳細介紹與評量架構說明書

---

## 壹、 系統整體簡介與設計理念

「**學生報告評量系統**」是一套專為**專題導向學習（Project-Based Learning, PBL）**與**期末成果報告發表**而打造的多元評量與學習成就回饋系統。

傳統的小組報告評量常面臨三個教學現場挑戰：
1. **單一主觀性**：僅由教師在台下打分，難以看見各小組內部成員間的實際分工與隱性付出。
2. **缺乏互助激勵**：小組報告易淪為少數「超級組員」全包，或出現「搭便車（Free-riding）」現象。
3. **質性與量化分離**：僅有分數缺乏具體稱讚與改善建議，或質性回饋過於零散不易統整。

本系統透過**「遊戲化闖關體驗（三關卡）」**，巧妙融合：
- **自我評量（Self-Assessment）**：學生後設認知（Metacognition）與自我省思。
- **同儕評量（Peer-Assessment）**：組內欣賞協作與閃光時刻驗證。
- **跨組標竿學習（Cross-Group Appreciation）**：他組星探發掘與互相觀摩。
- **專家教師評定（Expert Assessment）**：教師全域視角綜合把關。

實現了**「量化計分嚴謹公正、質性回饋具體溫暖」**的 360 度立體評量架構。

## 貳、 三大闖關關卡與核心評量向度

系統共設計三個循序漸進的關卡，每關分別對應不同的學習目標與評量規準（Rubrics）：

```
+-----------------------------------------------------------------------------------+
|                        學生報告評量系統 — 多元評量架構                            |
+-----------------------------------------------------------------------------------+
|  [關卡一：學生自評表]         [關卡二：組內互評表]         [關卡三：他組星探推薦]   |
|  (個人反思與表現自評)         (團隊協作與優勢激勵)         (跨組觀摩與鑑賞表揚)   |
|  - 7項專題表現五點量表        - 6大團隊超能力勳章          - 簡報吸睛王推薦       |
|  - 自給角色榮譽勳章           - 雙勳章加分 (3~10分)        - 金頭腦出題王推薦     |
|  - 2大核心反思與心得          - 具體閃光時刻質性描述       - 熱情神關主推薦       |
|                               - 隱藏 MVP 票選與真情告白                           |
+-----------------------------------------------------------------------------------+
```

### 【關卡一：學生自評表】（個人表現與反思評量）
- **評量目的**：培養對自身任務參與度、專題素養與反思表達的客觀審視能力。
- **量化向度（五點量表，7 大題，原始滿分 35 分）**：
  涵蓋 PBL 專題研究從起始到發表的 7 個核心環節：
  1. **資料蒐集**：使用可信來源、整理內容資訊
  2. **資料整理**：去蕪存菁、用自己的文字產出（避免抄襲）
  3. **小組討論**：尊重發言、積極貢獻想法
  4. **簡報討論**：努力貢獻心力、尊重組員想法、共同呈現最佳內容
  5. **簡報製作**：盡力完成負責部份、適時支援與協助其他組員
  6. **口頭報告**：有條有理、抑揚頓挫、音調與台風適中
  7. **解謎與活動**：攤位與活動當天堅守崗位、機動支援
- **評量規準（5 級分行為特徵描述）**：
  - **5 分（積極用心）**：全力以赴、表現卓越，不僅高品質達成職責，更能主動引導並協助同伴。
  - **4 分（表現良好）**：態度認真、穩定可靠，準時且落實完成所有工作與討論標準。
  - **3 分（表現平平）**：達成基本門檻，但在主動性、表達或協助方面尚有進步空間。
  - **2 分（仍需努力）**：被動參與，偶有遲交或工作完成度不足，需組員提醒與敦促。
  - **1 分（勉強做到）**：參與度明顯偏低，未能充分承擔相應責任。
- **質性與反思向度**：
  - **榮譽自我定位**：從 6 大勳章中選擇一枚最適合自己本學期表現的勳章。
  - **壓軸大哉問（引導式反思，嚴格檢驗字數下限以保證品質）**：
    - 問題 1：整個課程中，我最喜歡的部分是？（引導情感與興趣連結）
    - 問題 2：透過課程學習，我覺得我學到了哪些能力？（引導核心素養認知）

---

### 【關卡二：組內互評表】（團隊協作與優勢鑑賞）
- **評量目的**：轉化傳統互評容易出現的「比較與傷害」，引導學生以「尋找團隊超能力」的賞識心態，具體表揚每位同伴的特質。
- **量化與質性結合機制**：
  - **雙勳章與加分系統**：
    - 學生對同一組內的「每一位組員」皆可給予一枚「必填第一勳章」與一枚「選填第二勳章」。
    - 每枚勳章可對應給予 **3 分、4 分或 5 分** 的實質加分。
  - **六大團隊超能力勳章與特質定位**：
    1. **資料淘金客**：超會查資料、抓重點（資訊蒐集與探究專長）
    2. **進度小鬧鐘**：小組時空旅人，提醒截止時間（時程管理與進度掌控）
    3. **視覺魔法師**：簡報美化與排版救星（美感設計與資訊圖表排版）
    4. **氣場麥克風**：上台發表清楚、不怯場（口頭發表與現場溝通）
    5. **腦洞發明家**：點子王，解謎題目超有創意（創任意見與活動策劃）
    6. **溫暖神補位**：團隊黏著劑，默默撿起沒人做的事（後勤支援與互助同理）
  - **實證佐證（具體閃光時刻質性描述）**：
    - 每給予一位同學評價，**務必填寫至少 5 個字的「具體閃光時刻」**。
    - 範例：「在製作簡報時，他主動幫大家校對所有錯別字和格式，非常細心。」
- **隱藏大招：組內 MVP 投票與真情告白**：
  - 每人推舉一位全組公認的「本組年度隱藏 MVP」。
  - 填寫「想對 MVP 說的一句感謝」。
  - 獲選為 MVP 者，將獲得教師後台設定的 **MVP 額外加分**（預設 +1 分，教師可調整）。

---

### 【關卡三：他組星探推薦】（跨組標竿鑑賞與互動）
- **評量目的**：引導學生在活動當天走訪各組成果展示，成為「星探」，發掘別組表現傑出的楷模。
- **三大星探榮譽獎項**：
  1. **簡報吸睛王**（台上的靈魂人物）：推薦理由引導詞「我推薦... 因為他報告時：」
  2. **金頭腦出題王**（解謎題目的設計鬼才）：推薦理由引導詞「我推薦... 因為解他們題目時：」
  3. **熱情神關主**（活動當天的最佳服務生）：推薦理由引導詞「我推薦... 因為去他們攤位時：」
- **加分回饋**：榮獲別組同學星探推薦者，將獲得教師後台設定的 **星探推薦額外加分**（預設每次 +2 分，無上限累計或依教師規則）。

## 參、 質性與量化評量融合分析模型

本系統的設計精髓在於讓 **「數字有依據，文字有溫度」**：

| 評量向度 | 量化機制 (Quantitative Indicators) | 質性機制 (Qualitative Evidences) |
| :--- | :--- | :--- |
| **自我評量<br>(Self-Assessment)** | • 7 題五點量表評分（滿分 35 分）<br>• 自動依公式換算為 100 分制百分比 | • 自給勳章定位（優勢自我認同）<br>• 最喜歡的課程環節反思<br>• 學到的能力自我盤點 |
| **組內互評<br>(Peer-Assessment)** | • 組員獲贈勳章加分總和（每位組員給出 3~10 分）<br>• 組內 MVP 得票數乘以單票權重加分 | • 給予每位組員的具體閃光時刻描述<br>• 給予組內 MVP 的致謝告白 |
| **跨組鑑賞<br>(Cross-Assessment)** | • 獲得他組星探推薦總次數乘以單次加分 | • 簡報吸睛王 / 出題王 / 神關主之具體推薦理由 |
| **教師專家評定<br>(Expert-Assessment)** | • 教師綜合給分（0 ~ 100 分） | • 後台全域掌握全班各小組與個人的量化質性表現 |

## 肆、 綜合成績運算公式與權重機制

系統教師後台支援 **自訂權重參數**，所有分數自動即時計算，免除人工統計算分誤差：

### 1. 各項分數轉換計算
- **自評百分制（Self Score 100-scale）**：
  $$\text{自評百分制} = \left(\frac{\text{自評 7 題原始得分累計}}{35}\right) \times 100$$
- **他評加分總和（Peer Bonus Score）**：
  $$\text{他評加分總和} = \sum(\text{組內成員給予之勳章分數}) + (\text{MVP得票數} \times \text{MVP每票加分}) + (\text{他組星探推薦次數} \times \text{星探每次加分})$$
- **老師評分（Teacher Score）**：
  教師根據現場報告與書面專題表現評定的分數（0 ~ 100 分）。

### 2. 最終綜合成績計算公式
預設權重為：**自評佔 30%、他評加分總和佔 20%、老師評分佔 50%**（比例皆可在後台隨時自由自訂並儲存）：

$$\text{最終綜合分數} = \left(\text{自評百分制} \times \frac{\text{自評權重}}{100}\right) + \left(\text{他評加分總和} \times \frac{\text{他評權重}}{100}\right) + \left(\text{老師評分} \times \frac{\text{老師權重}}{100}\right)$$

## 伍、 教師專屬後台與評量報表應用

系統提供完整功能與介面的教師專屬後台（Teacher Admin），主要分為五大模組：
1. **成績結算與打分**：統整全班座號、組別、各向度得分與最終綜合評量成績，供老師在此直接輸入打分。
2. **學生填寫紀錄（輸出明細）**：完整查看每位學生填寫的所有自評分數、兩大質性反思、給予同組成員的勳章分數與閃光時刻、MVP 推舉與理由。
3. **學生收到評價（獲得回饋明細）**：彙整各同學被同組夥伴給予的所有讚美勳章、分數、閃光時刻留言、獲選 MVP 感謝與他組推薦。
4. **名單管理**：支援 1~6 組成員座號增刪修與雲端同步。
5. **計分權重設定**：可自由調整自評、他評、老師評分百分比與 MVP/星探單票加分標準，亦可自訂學生首頁登入密碼。

### 📊 完整報表匯出功能（CSV / Excel 雙格式）
系統支援一鍵匯出 **UTF-8 BOM 編碼之完整 CSV 總表** 與 **Excel 雙工作表 (.xlsx)**，報表含 **28 個全面性欄位**，適合教學評審、成績匯入系統及期末歷程檔案留存：
- **基本資訊與分數**：座號、組別、最終綜合分數、老師綜合評分、自評百分制、他評加分總和、MVP得票數、自給勳章。
- **自評量化與質性明細**：自評總分（滿分35）、第 1~7 題各題給分（1~5分）、最喜歡的部分、學到的能力。
- **他評與推薦輸入明細**：對同組各成員之互評紀錄（對象/勳章/給分/閃光時刻）、推舉 MVP 座號與感謝文字、星探推薦吸睛王/出題王/神關主與理由。
- **收到的所有評價明細**：收到的組內勳章與分數總覽、收到的閃光時刻留言明細、收到的 MVP 感謝與他組星探推薦理由。

## 陸、 學生首頁登入驗證與測試帳號機制

1. **一般學生登入（雙重驗證）**：
   - 學生需輸入「座號（1~30號）」與「登入密碼（預設為 `c1723`，教師可在後台自訂修改）」方能進入任務總部。
2. **專屬體驗測試帳號（31 號）**：
   - 於座號欄輸入 **`31`** 號為系統體驗與測試帳號，**無須登入密碼**即可直接進入。
   - 測試帳號填寫的任何自評或互評資料**皆不會上傳至資料庫，亦不會列入教師後台的成績與報表紀錄**。
